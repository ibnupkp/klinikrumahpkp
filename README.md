# Klinik Rumah PKP — Konsultasi Desain Rumah

Repo ini akan diisi otomatis oleh ChatGPT Codex untuk membangun fitur:
- Prototype 22/27/36 m² (sesuai Kepmen PUPR 2947/2024) → unduh paket **DED & RAB (.zip)**
